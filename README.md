# space_hk
Keep all the co-working spaces in one place

There are 60+ co-working spaces in Hong Kong, some of them are really new. This repo aims to curate those spaces in one place.

- Venue
- Address
- Contact Information
- Promotion
- Events
- Startup Community
- Startups working in there
